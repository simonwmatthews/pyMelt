Installation
^^^^^^^^^^^^
It is possible to install pyMelt using pip:

.. code-block:: python

   pip install pyMelt

Always use the most up-to-date version of the code:

.. code-block python

   pip install pyMelt --upgrade

See the installation section for detailed instructions and dependencies.
